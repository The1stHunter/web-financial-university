Задание t-2

Условие

В HTML-документе определены 2 формы.
На первой управляющие кнопки и поля ввода, вторая - пустая.
При нажатии на кнопки первой формы на второй форме генерируются элементы TABLE из одного строки, несколько флажков (по числу ячеек).
Параметры и количество элементов определяются в элементах 1-й формы. В элементе INPUT выбрать число ячеек, а также заданы в виде массивов на JavaScript.
После построения элементов на второй форме по некоторым кнопкам/элементам можно произвести некоторые действия (при отметке флажка значение соответствующей ячейки инвертируется Значения ячеек заданы в массиве), результаты которых отображаются на 2-форме.
Для генерации элементов и вывода результатов использовать свойство innerHTML.
Примечание. Уточнения постановки производить самостоятельно.
